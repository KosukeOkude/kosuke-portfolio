pt（上のみ）・pb（下のみ）
pt-◯ と pb-◯ は上記と同じ数値で、それぞれ padding-top / padding-bottom のみ指定します。

例：

pt-4 → padding-top: 1rem;
pb-4 → padding-bottom: 1rem;
60px にしたい場合
Tailwind のスケールでは py-15 (60px) が使えます。任意値なら py-[60px] も可能です。


Tailwind での指定（任意ブレークポイント）
Tailwind の sm: (640px) より小さいブレークポイントはプリセットにないので、ブラケットで指定します。

390px 以上で表示する場合：

<p class="... hidden min-[390px]:block">
  <slot />
</p>
400px 以上で表示する場合：

<p class="... hidden min-[400px]:block">
  <slot />
</p>
hidden … デフォルト（〜389px or 〜399px）は非表示
min-[390px]:block or min-[400px]:block … その幅以上で表示



了解です。CSS の理屈はわかっている前提で、「CSS → Tailwind クラス」の対応だけをザッと列挙しますね。

基本レイアウト
display: flex; → flex
display: inline-flex; → inline-flex
flex-direction: row; → flex-row（デフォルト）
flex-direction: row-reverse; → flex-row-reverse
flex-direction: column; → flex-col
flex-direction: column-reverse; → flex-col-reverse

主軸揃え（justify-content）
justify-content: flex-start; → justify-start
justify-content: flex-end; → justify-end
justify-content: center; → justify-center
justify-content: space-between; → justify-between
justify-content: space-around; → justify-around
justify-content: space-evenly; → justify-evenly

交差軸揃え（align-items）
align-items: flex-start; → items-start
align-items: flex-end; → items-end
align-items: center; → items-center
align-items: baseline; → items-baseline
align-items: stretch; → items-stretch

複数行レイアウトの揃え（align-content）
align-content: flex-start; → content-start
align-content: flex-end; → content-end
align-content: center; → content-center
align-content: space-between; → content-between
align-content: space-around; → content-around
align-content: space-evenly; → content-evenly
align-content: stretch; → content-stretch

個別アイテム揃え（align-self）
align-self: auto; → self-auto
align-self: flex-start; → self-start
align-self: flex-end; → self-end
align-self: center; → self-center
align-self: stretch; → self-stretch
align-self: baseline; → self-baseline

折り返し（flex-wrap）
flex-wrap: nowrap; → flex-nowrap
flex-wrap: wrap; → flex-wrap
flex-wrap: wrap-reverse; → flex-wrap-reverse

ギャップ（row-gap / column-gap / gap）
gap: 1rem; → gap-4（数値は Tailwind のスケールに依存）
column-gap: …; → gap-x-*
row-gap: …; → gap-y-*

フレックスアイテムのサイズ・伸縮
flex-grow: 0; → grow-0
flex-grow: 1; → grow
flex-shrink: 0; → shrink-0
flex-shrink: 1; → shrink
flex: 1 1 0%;（よくある「全部等分」）→ flex-1
flex: none; → flex-none
flex: 0 1 auto; → flex-initial
flex-basis: 50%; → basis-1/2
flex-basis: 25%; → basis-1/4
flex-basis: auto; → basis-auto
任意値の flex-basis → basis-[200px] など任意値記法も可
並び順（order）
order: 0; → order-none（実際は order-0 もある）
order: 1; → order-1
order: 2; → order-2
… order-12 まで
特別系 → order-first, order-last
